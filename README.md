## Running BanglaScript
To execute a BanglaScript file, use the interpreter:

```bash
bs <file_path>
```

### Example:
If your script is named `main.bs`, run:
```bash
bs main.bs
```

## Examples
### Example 1: Basic Variable Assignment and Printing
**main.bs**:
```
dhoro x = 5
lekho(x)
```
**Output**:
```
5
```

### Example 2: Printing Direct Values
**main.bs**:
```
lekho(10)
```
**Output**:
```
10
```
